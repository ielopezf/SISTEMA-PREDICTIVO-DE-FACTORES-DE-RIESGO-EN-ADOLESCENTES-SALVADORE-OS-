# SISTEMA-PREDICTIVO-DE-FACTORES-DE-RIESGO-EN-ADOLESCENTES-SALVADORE-OS-

SISTEMA PREDICTIVO DE FACTORES DE RIESGO EN ADOLESCENTES SALVADOREÑOS MEDIANTE TÉCNICAS DE MACHINE LEARNING BASADO EN LA ENCUESTA GSHS 2013

Proyecto desarrollado para el Laboratorio Práctico 2 de la Especialización en Machine Learning de la Universidad de El Salvador (UES).

Especialización en Machine Learning

Universidad de El Salvador

**Autores**

- Irma Elena López Fuentes
- Elena Sarai Serrano Fuentes

El proyecto implementa un pipeline completo de Machine Learning utilizando los datos de la Global School-based Student Health Survey (GSHS) El Salvador 2013, con el objetivo de apoyar la identificación temprana de factores de riesgo en adolescentes mediante modelos predictivos.

# Modelos implementados

### Regresión

- Regresión Lineal
- Random Forest Regressor

### Métricas utilizadas

- RMSE
- R²

### Clasificación

- Regresión Logística
- Random Forest Classifier

### Métricas utilizadas

- Accuracy
- Precision
- Recall
- F1-Score
- ROC-AUC
